# Power BI Dashboard

This folder contains the Power BI dashboard used to analyse the loan portfolio.

## 📊 Dashboard Overview

The dashboard provides insights into:

- Default rate across borrower segments
- Credit grade risk distribution
- Loan amount vs interest rate relationship
- Portfolio exposure analysis
